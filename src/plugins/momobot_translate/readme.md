# momobot_translate
a plugin for nonebot2

## 使用
1. 前往百度翻译开放平台
自行申请[百度翻译api](https://api.fanyi.baidu.com/api/trans/product/index)
2. 修改`config.py`中数据;

    ``` python
    # /config.py
    appid = "xxxxxxxx"  # 填写你的appid
    secretKey = "xxxxxxxxx"  # 填写你的密钥
    ```


## 功能
| 指令 | 功能 |
| :- | :- |
| 翻译 <要翻译的词句> | 自动中译英，其他语种译中 |
